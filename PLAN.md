## 造梦西游3 复刻版 — 分阶段开发计划

### 技术栈

| 层 | 技术 |
|---|---|
| 游戏引擎 | Phaser 3 + TypeScript |
| 构建工具 | Vite |
| 后端 | Spring Boot + MyBatis Plus |
| 数据库 | MySQL |
| 认证 | JWT |
| 缓存 | Redis（后期按需） |

---

### 第一阶段：能跑能跳（核心操作手感）

> 目标：一个角色在场景里自由移动、跳跃、攻击，打死几只小怪。这是最基础也最重要的一步——手感对了，后面一切都顺。

1. **项目脚手架搭建**
   - Phaser 3 + TypeScript + Vite 工程初始化
   - 目录结构规划（scenes / entities / assets / config）
   - 开发热更新 & 调试配置

2. **游戏场景基础**
   - 主菜单场景（开始游戏按钮）
   - 战斗场景加载，横版卷轴地图（静态背景 + 地面碰撞体）
   - 相机跟随角色

3. **角色基础操作**
   - 左右移动（加速度 & 摩擦力，手感调优）
   - 跳跃（重力系统，二段跳预留）
   - 朝向翻转（左右-facing）
   - 动画状态机：待机 → 跑动 → 跳跃 → 下落

4. **基础战斗**
   - 普通攻击（近战挥砍，攻击判定框 / hitbox）
   - 攻击连段（轻攻击 × 3 连击）
   - 受击反馈（击退 + 闪白 + 硬直）

5. **简单怪物**
   - 一种小怪：巡逻 AI（左右走动，遇边缘翻转）
   - 怪物受击 & 死亡
   - 血条显示（角色 + 怪物头顶 HP 条）

**阶段交付物**：能操控悟空在一张地图里跑来跑去砍死小怪的 Demo。

---

### 第二阶段：像模像样（RPG 系统骨架）

> 目标：加入技能、多怪物种类、关卡流程，让游戏开始有"造梦西游"的味道。

6. **技能系统**
   - MP 蓝条 & 消耗机制
   - 2~3 个主动技能（如：烈焰闪、火魔斩、七十二斩）
   - 技能冷却（CD 显示）
   - 技能特效（粒子 / 动画）

7. **怪物丰富化**
   - 2~3 种不同行为的小怪（近战型、远程投掷型、冲锋型）
   - 怪物刷新波次系统
   - 简单的 BOSS：固定攻击模式（冲锋 + 跳砸 + 弹幕）

8. **关卡流程**
   - 波次推进：清完一波刷下一波
   - 关卡胜利（全清）& 失败（角色死亡）
   - 结算界面（经验、金币）

9. **UI 系统**
   - HUD（血条、蓝条、技能图标、CD 遮罩）
   - 伤害飘字
   - 暂停菜单

**阶段交付物**：一个完整可玩的关卡，有技能释放、多种怪物、有 BOSS 战。

---

### 第三阶段：有内味了（成长 & 装备系统）

> 目标：让角色能变强、能换装备，有 RPG 的养成乐趣。

10. **角色属性系统**
    - 基础属性（攻击、防御、暴击、生命、魔法）
    - 等级 & 经验值曲线
    - 升级属性增长

11. **装备系统**
    - 武器 / 防具 / 饰品三个槽位
    - 装备属性加成
    - 装备品质（白 → 绿 → 蓝 → 紫 → 橙）
    - 背包 UI

12. **关卡选择**
    - 大地图 / 关卡列表
    - 多关卡（3~5 关，难度递增）
    - 关卡星级评价

13. **数据持久化（本地）**
    - localStorage 存档 / 读档
    - 角色状态、装备、关卡进度保存

**阶段交付物**：有成长循环的完整游戏体验——打怪 → 升级 → 换装备 → 挑战更难的关卡。

---

### 第四阶段：联网 & 账号（后端接入）

> 目标：加入用户系统和云存档，让游戏有"产品"的感觉。

14. **Spring Boot 后端搭建**
    - 项目初始化（Spring Boot + MyBatis Plus + MySQL）
    - 数据库表设计（用户、角色存档、装备、关卡记录）
    - JWT 认证（注册 / 登录 / Token 刷新）

15. **前后端联调**
    - 前端登录 / 注册界面
    - 云存档同步（上传 / 下载）
    - API 接口对接

16. **排行榜**
    - 关卡评分排行
    - 战力排行
    - Redis 缓存热门排行数据

**阶段交付物**：有账号系统、云存档、排行榜的完整 Web 游戏。

---

### 第五阶段：锦上添花（可选扩展）

> 按兴趣和精力选做，不强求。

17. **多角色**：猪八戒 / 沙僧 / 唐僧，不同技能组
18. **宠物 / 坐骑系统**
19. **音效 & BGM**：战斗音效、背景音乐
20. **移动端适配**：虚拟摇杆 + 触屏按钮
21. **PVE 挑战模式**：无尽塔、生存模式
22. **美术升级**：更精致的精灵图和特效

---

### 关于美术资源的说明

前期开发阶段不必纠结原创美术，可以用以下策略：

- 使用开源 / 免费的 2D 像素或手绘素材（itch.io、kenney.nl 等）
- 用简单的几何图形 placeholder 先跑通逻辑
- AI 生成辅助素材（背景、UI 元素等）
- 后期再考虑定制美术
