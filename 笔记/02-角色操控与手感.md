## 角色操控 & 游戏手感

### Container + Graphics（修复蓝框偏移 bug）
- 之前用 `Rectangle + setScale(-1,1)` 翻转朝向，物理体不会跟着翻，导致蓝框错位
- 修复方案：用 `Container`（容器）包裹 `Graphics`（绘图）
  - Container 负责位置和物理
  - Graphics 负责画面，重绘即可翻转，不需要 setScale
- 以后换精灵图，只需要改 `drawPlayer()` 方法

### 二段跳
- 用 `jumpCount` 记录跳了几次，落地时归零
- `jumpCount < 2` 就允许再跳 → 实现二段跳
- `JustDown(space)` = 只在按下的那一帧触发，不是按住就连续跳

### 双击冲刺
- 记录上次按同一方向键的时间 `lastTapTime`
- 两次间隔 < 300ms → 判定为双击 → 触发冲刺
- 冲刺期间关闭摩擦力 `setDragX(0)`，保持高速
- 冲刺结束后恢复摩擦力，自然减速

### 按键检测的区别
```ts
key.isDown        // 持续按住（每帧都检测）
JustDown(key)     // 只在按下的那一帧为 true（适合跳跃、攻击这种"按一下触发一次"的操作）
key.on("down")    // 事件监听（适合双击检测这种需要精确时间的场景）
```

### this.time.now
- Phaser 内置时间戳（毫秒），不受系统时钟影响
- 适合做冷却、倒计时、双击判定窗口
