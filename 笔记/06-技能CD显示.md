# 06 - 技能冷却（CD）显示系统

## 核心思路

技能 CD 显示 = 每帧检查剩余冷却时间 + 可视化遮罩 + 倒计时数字。

### 1. 数据结构

每个技能只需要一个时间戳：

```typescript
private skill1CooldownEnd = 0;  // 技能1冷却结束的绝对时间
private skill2CooldownEnd = 0;  // 技能2冷却结束的绝对时间
```

用 `time.now < skillCooldownEnd` 判断是否在 CD 中。
用 `skillCooldownEnd - time.now` 计算剩余毫秒数。

### 2. 使用技能时记录 CD

```typescript
// 消耗 MP，进入 CD
this.playerMp -= this.SKILL1_MP_COST;
this.skill1CooldownEnd = now + this.SKILL1_COOLDOWN;
```

### 3. 可视化 CD 图标

每个技能图标是一个 Container，包含：
- 底色矩形（深色，让遮罩叠加更明显）
- 彩色内圈（技能代表色）
- 快捷键字母（左上角）

CD 中的显示：
- 半透明黑色 Rectangle 覆盖图标（变暗）
- 白色文字显示剩余秒数（Math.ceil 向上取整）

MP 不足的显示：
- 更浅的遮罩（提示不可用）
- 不显示倒计时（因为没有 CD，只是缺蓝）

### 4. 每帧刷新（updateSkillCdDisplay）

```
if (剩余CD > 0)       → 显示遮罩 + 倒计时
else if (MP < 消耗)   → 显示浅遮罩（缺蓝）
else                  → 隐藏遮罩（技能就绪）
```

## 关键知识点

- `Math.ceil(ms / 1000)` 把毫秒转成"剩余秒数"（向上取整，5001ms → 6秒）
- `setFillStyle(color, alpha)` 可以动态改变矩形的透明度
- `setVisible(true/false)` 比 `setAlpha(0)` 更高效（不可见时不参与渲染）
- 图标的 `setScrollFactor(0)` 让 HUD 固定在屏幕上

## 可调参数

| 参数 | 值 | 说明 |
|------|-----|------|
| SKILL1_MP_COST | 20 | 烈焰闪耗蓝 |
| SKILL1_COOLDOWN | 5000ms | 烈焰闪CD |
| SKILL2_MP_COST | 30 | 巨剑术耗蓝 |
| SKILL2_COOLDOWN | 8000ms | 巨剑术CD |
