## 攻击系统 & 敌人

### 连招系统
- `comboStep` 记录当前第几段（1/2/3）
- `comboTimer` 记录连招窗口截止时间
- 在窗口内再按 J → 接下一段；超时 → 重置回第一段
- 三段伤害递增：10 → 15 → 25，击退也递增

### 攻击判定（hitbox）
- 攻击时在角色面朝方向创建一个半透明矩形
- 用 `physics.overlap()` 检测矩形和敌人组是否重叠
- 攻击结束后 `hitbox.destroy()` 销毁判定框
- 每次攻击只命中一次（hitbox 存活时间短）

### 受击反馈三件套
1. **击退** → `body.setVelocityX(knockbackX)`
2. **闪白** → Tween 快速 alpha 0.3 → 1（yoyo + repeat）
3. **伤害飘字** → 文字 Tween 上飘 + 淡出后销毁

### Tween（补间动画）
```ts
this.tweens.add({
  targets: 目标对象,
  alpha: 0,        // 属性变化终点
  duration: 600,   // 毫秒
  yoyo: true,      // 播完倒着播回来
  repeat: 1,       // 重复次数
  onComplete: () => { ... }  // 结束回调
});
```
- Tween 适合做短暂的视觉效果（闪烁、飘字、缩放、旋转）
- 不需要手动在 update 里写插值，Phaser 自动处理

### 敌人管理
- 用 `staticGroup` 统一管理所有敌人（方便碰撞检测）
- 每个敌人用 `container.setData("enemy", enemyData)` 存数据
- 碰撞回调中通过 `getData("enemy")` 取出数据来操作

### this.physics.overlap()
- 检测两个物理体是否重叠（不像 collider 会弹开）
- 适合做"判定框碰到了敌人"这种一次性检测
- 回调参数：`(对象A, 对象B) => { ... }`
