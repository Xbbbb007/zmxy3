# 09 - 小怪设计

## 概述

小怪是非 BOSS 类敌人的统称，包括三种类型：普通近战怪（NORMAL）、冲锋怪（CHARGER）、远程投掷怪（THROWER）。

小怪的共同特征：前摇可打断、受击完整硬直（300ms）、全额击退、死亡动画旋转升天。

代码位于 `game/src/enemies/minions/` 文件夹，从 EnemyManager 中独立出来。

## 代码结构

```
enemies/
  EnemyManager.ts          ← 总协调（生成/调度），BOSS 逻辑也在这里
  minions/
    index.ts               ← 统一导出
    MinionConfig.ts        ← 所有小怪参数配置（改数字调难度）
    MinionBehavior.ts      ← 共享行为（巡逻、受击、死亡、画外观、血条）
    NormalAI.ts            ← 普通近战怪 AI（前摇/攻击/冷却）
    ChargerAI.ts           ← 冲锋怪 AI（蓄力/冲锋/碰撞）
    ThrowerAI.ts           ← 投掷怪 AI（瞄准/发射/后退）
```

## 普通近战怪（NORMAL）

**定位**：基础敌人，给玩家练手，威胁低，容易预判。

### 外观

蓝灰色方块（32x48px），白色小眼睛 + 黑色瞳孔。

### AI 行为

1. 巡逻：在出生点 ±120px 范围内以 60px/s 速度左右走动
2. 感知：玩家进入 80px 才注意到
3. 前摇：站定 500ms，头顶红色 `!` 闪烁（可被打断）
4. 攻击：近战挥砍，判定范围 45px，伤害 10，击退 180
5. 冷却：攻击后 800ms 恢复巡逻

### 参数表

| 参数 | 值 | 说明 |
|------|-----|------|
| PATROL_SPEED | 60 | 巡逻速度 |
| PATROL_RANGE | 120 | 巡逻范围（出生点 ±） |
| DETECT_RANGE | 80 | 感知距离 |
| ATTACK_WINDUP | 500ms | 前摇时间 |
| ATTACK_DURATION | 200ms | 攻击判定持续 |
| ATTACK_COOLDOWN | 800ms | 冷却时间 |
| ATTACK_DAMAGE | 10 | 伤害 |
| ATTACK_RANGE | 45px | 攻击距离 |
| ATTACK_KNOCKBACK | 180 | 击退力度 |

### 视觉反馈

- 前摇：红色 `!` 感叹号闪烁
- 攻击：红色弧线特效（strokeEllipse）

## 冲锋怪（CHARGER）

**定位**：中距离高威胁敌人，远距离发现玩家后蓄力冲撞。玩家需要在前摇期间远离或打断。

### 外观

绿色宽扁方块（40x40px），两只黄色角（像野牛），红色眼睛。

### AI 行为

1. 巡逻：和普通怪一样，120px 范围，60px/s
2. 感知：250px 远距离发现（比普通怪远 3 倍）
3. 前摇：站定 600ms，头顶橙色 `!!` 闪烁（可打断）
4. 冲锋：以 450px/s 高速冲向面朝方向，沿途碰撞检测
5. 冷却：冲锋结束后 1200ms 恢复巡逻

### 参数表

| 参数 | 值 | 说明 |
|------|-----|------|
| DETECT_RANGE | 250 | 感知距离 |
| CHARGE_SPEED | 450 | 冲锋速度 |
| CHARGE_WINDUP | 600ms | 蓄力时间 |
| CHARGE_DURATION | 500ms | 冲锋持续 |
| CHARGE_DAMAGE | 15 | 冲撞伤害 |
| CHARGE_KNOCKBACK | 250 | 冲撞击退 |
| COOLDOWN | 1200ms | 冲锋后冷却 |

### 视觉反馈

- 前摇：橙色 `!!` 双感叹号
- 冲锋起点：3 颗灰色尘土粒子向上飘散

## 远程投掷怪（THROWER）

**定位**：远程骚扰型，玩家要顶着弹幕靠近才能打。血量低但会跑。

### 外观

紫色瘦高方块（24x56px），右手持黄色法杖（带圆形宝石），青色眼睛。

### AI 行为

1. 巡逻：120px 范围，60px/s
2. 感知：200px 发现玩家
3. 特殊行为：玩家距离 < 80px 时后退（80px/s），保持距离
4. 前摇：站定 400ms，头顶紫色 `✦` 星号闪烁（可打断）
5. 瞄准：法杖闪光 200ms
6. 发射：向玩家当前位置发射紫色能量球
7. 冷却：1500ms

### 参数表

| 参数 | 值 | 说明 |
|------|-----|------|
| DETECT_RANGE | 200 | 感知距离 |
| PREFERRED_DIST | 150 | 偏好攻击距离 |
| AIM_TIME | 400ms | 瞄准时间 |
| COOLDOWN | 1500ms | 发射后冷却 |
| PROJECTILE_SPEED | 350 | 投掷物速度 |
| PROJECTILE_DAMAGE | 8 | 投掷物伤害 |
| RETREAT_SPEED | 80 | 后退速度 |
| RETREAT_THRESHOLD | 80px | 太近触发后退 |

### 视觉反馈

- 前摇：紫色 `✦` 星号闪烁
- 瞄准：法杖位置黄色圆形放大淡出
- 投掷物：紫色外圈 + 黄色内核高光跟随
- 3 秒后自动销毁

## 共享机制

### 巡逻（Patrol）

所有小怪共用的巡逻逻辑：在出生点附近左右走动。被击退时暂停巡逻（检测速度 < 5px/s），等摩擦力（dragX=400）把速度衰减完再恢复。

### 受击（Hit）

小怪被打后的统一处理：
1. 如果在 WINDUP 状态 → 打断前摇（感叹号消失）
2. 进入 HIT 状态 300ms（完整硬直）
3. 全额击退
4. 闪烁（alpha 0.3 → 1，60ms，yoyo × 2）
5. 伤害飘字（红色，上飘淡出）

### 死亡（Kill）

统一死亡动画：旋转 360° + 上飘 30px + 淡出，400ms 后销毁容器和血条。

### 血条

每个小怪头顶有 38px 宽的血条。HP > 30% 为绿色，≤ 30% 变红色。

### 碰撞体对齐

所有小怪的碰撞体底部对齐（站在同一条地面上），但宽度根据体型不同：
- NORMAL: 32x48
- CHARGER: 40x40
- THROWER: 24x56

---

## 设计意图

三种小怪形成一个简单的"敌人三角"：

- **普通怪**：近距离、低威胁 → 练手
- **冲锋怪**：中距离、高爆发 → 需要反应和走位
- **投掷怪**：远距离、持续骚扰 → 需要快速接近

这迫使玩家在不同距离上采取不同策略，为后续的关卡设计和波次组合打下基础。
