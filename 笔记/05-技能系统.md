## 技能系统

### MP（蓝条）
- 和 HP 一样的条状显示，蓝色
- 自然恢复：每秒 +5 MP
- 公式：`playerMp += MP_REGEN * (delta / 1000)` — delta 是帧间隔，保证不按帧率恢复

### 技能数据结构
```ts
// 每个技能需要定义：
private readonly SKILL1_MP_COST = 20;      // MP 消耗
private readonly SKILL1_COOLDOWN = 5000;   // 冷却(ms)
private readonly SKILL1_DAMAGE = 30;       // 伤害
// ...其他参数
```

### 烈焰闪（K 键）
- 消耗 20 MP，5 秒 CD
- 效果：向前冲刺 200px，路径上的敌人受到 30 火焰伤害
- 冲刺期间不受攻击锁定影响
- 视觉：火焰拖尾粒子

### 技能释放检查顺序
1. MP 够不够 → `playerMp >= MP_COST`
2. CD 好没好 → `now >= cooldownEnd`
3. 状态允不允许 → `!isAttacking && !isDashing`

### this.game.loop.delta
- 上一帧到这一帧的时间差（毫秒）
- 用它可以做"与帧率无关"的计算（比如 MP 恢复速度）
- 如果不用 delta，60fps 和 30fps 下恢复速度会不一样
