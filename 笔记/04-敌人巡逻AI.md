## 敌人巡逻 AI

### 基本思路
- 每个敌人出生时记录 `patrolLeft` 和 `patrolRight`（出生点 ± 巡逻范围）
- 每帧检测敌人位置，走到边界就掉头（`facingRight` 取反）
- 被击退时（速度较大）暂停巡逻，等停下来再继续

### 关键代码逻辑
```ts
// 只有速度接近 0 时才巡逻（被击退时不干扰）
if (Math.abs(body.velocity.x) < 5) {
  if (enemy.facingRight) {
    body.setVelocityX(PATROL_SPEED);
    if (enemy.container.x >= enemy.patrolRight) enemy.facingRight = false;
  } else {
    body.setVelocityX(-PATROL_SPEED);
    if (enemy.container.x <= enemy.patrolLeft) enemy.facingRight = true;
  }
}
```

### 为什么检测速度 < 5 ？
- 被玩家攻击后，`setVelocityX(knockback)` 会给敌人一个很大的速度
- 此时如果强行覆盖成巡逻速度，击退效果就没了
- 等击退速度衰减到接近 0（靠 `dragX` 摩擦力），再恢复巡逻

### 可调参数
- `ENEMY_PATROL_SPEED = 60` → 巡逻速度，像散步一样慢
- `ENEMY_PATROL_RANGE = 120` → 巡逻范围，出生点左右各走 120px
